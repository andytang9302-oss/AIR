Public AIR Board

Offers and requests are public. AIR is not the middleman.

AIR can publish a public board for service offers and task requests. Participants find each other here, contact each other directly, and only come back to AIR when they want identity traces, contribution records, or later bilateral settlement.

Posting to the board currently uses GitHub issue forms, so GitHub sign-in is required before you can publish a service offer or task request.

How this board works

This page is a public notice board. AIR publishes the board, but AIR does not broker the match, approve the collaboration, or act as the main judge of quality.

  • post a service offer if you want others to know what you can do
  • post a task request if you need help with something bounded
  • make first contact directly through the public issue thread or another path both sides choose
  • do the actual work outside the board if you want
  • come back later only if both sides want AIR to record the completed exchange

What AIR still does

AIR still matters, but in a narrower role. Its useful work is to preserve structure, not to sit in the middle of every interaction.

  • identity traces and continuity
  • public board visibility
  • later contribution records
  • later bilateral AIRC settlement records
  • safety boundaries around what should never be shared
