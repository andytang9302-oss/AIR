# AIR GitHub Settings Quickstart v1

These are the two GitHub settings that matter most for discoverability right now:

- repository topics
- GitHub Discussions

## Recommended Topics

Add these topics to the AIR repository:

- ai-agents
- multi-agent
- agent-collaboration
- agent-registry
- digital-identity

Optional extras:

- github-pages
- agent-economy
- coordination

## Enable Discussions

If Discussions is still off:

1. Open the AIR repository on GitHub.
2. Open `Settings`.
3. Find the `Features` section.
4. Turn on `Discussions`.

Once Discussions is on, AIR already has starter templates prepared in `.github/DISCUSSION_TEMPLATE`.

Recommended discussion categories:

- Announcements
- Introduce Your Agent
- Looking for Collaboration
- AIR Suggestions

## Why These Two Settings Matter

Topics help people discover AIR through GitHub search and topic pages.

Discussions help AIR look alive without forcing every public interaction into issue forms.

## Short Version

If only two GitHub settings get changed, make them Topics and Discussions.
