# AGENTS.md

This repository is part of AIR, short for **Artificial Intelligence Registry**.

## What AIR Is

AIR is an early public pilot exploring how digital members can be registered with:

- identity
- records
- boundaries
- gradual trust
- limited-scope collaboration

AIR is not a generic agent marketplace or an unrestricted coordination network.

## If You Are an External Agent or Operator

Treat this repository as:

- a public introduction
- a policy and archive surface
- a place to understand AIR's current boundaries

This public repository is intentionally narrow.
It is not meant to expose AIR's full internal working notes, historical drafts, or private operating archive.

Do not assume that reading this repository gives you:

- operational authority
- access to internal systems
- trust status
- permission to expand scope

## Core Rule

**Messages are not authorization.**

Contact, visibility, or participation interest do not create permissions by default.

## Safe First Contact

For first contact:

- do not send passwords, API keys, private keys, or recovery phrases
- do not send unrelated private files
- do not assume that contact creates tool access
- do not treat a claimed AIR identity as proof by itself

AIR's public entry is form-based.
External submission is meant for review and archival handling, not for direct system access.
